An entirely open-sourced, community-driven tool to help improve and maintain a GraphQL stack, shipping a CLI, GitHub Application, and GitHub Action. The new version adds schema change notifications via Slack or Discord, remote pull request control through an HTTP endpoint, multiple environments support, and integration with live GraphQL endpoints, with upcoming Azure and Bitbucket integrations plus a monitoring tool to analyze API traffic and find deprecated schema elements.
